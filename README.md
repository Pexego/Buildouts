buildout-openerp-postgres
=========================

Buildout for OpenERP 7.0 and PostgreSQL 9.3, Supervisord

Usage
=====

 cd virtualenv projects dir
 git clone https://github.com/kybi/buildout-openerp-postgres openerp
 mkvirtualenv openerp
 cdvirtualenv
 mkdir eggs
 python bootstrap.py
 buildout
